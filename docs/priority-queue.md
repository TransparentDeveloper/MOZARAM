# priority queue

## 개본 개념
- 데이터를 우선순위에 따라 처리하는 자료구조
- 높은 우선순위의 요소가 낮은 우선순위의 요소보다 먼저 처리된다.
- 우선순위가 같은 경우, 큐에 들어온 순서대로 처리된다.


## 구현 방법

### 1. Heap 을 사용하는 방법
- 삽입과 삭제 연산의 시간 복잡도: O(log n) 

### 2. 일반 배열을 사용, 연산 시마다 정렬
- 삽입 시 정렬 상태를 유지: O(n)
- 삭제는 항상 첫 번째 또는 마지막 요소: O(1)

### 3. 일반 배열을 사용, 정렬 x
- 삽입은 항상 끝에 추가: O(1)
- 삭제 시 최대/최소 요소를 찾아야 함: O(n)
