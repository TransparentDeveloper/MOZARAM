# 소인수분해

## 요약 설명
- 주어진 임의의 자연수(n) 에 대하여, n 을 구성하는 모든 소수를 출력

<br/>

## 언제 사용하면 좋은가?
- 시간 복잡도: **O( √n )**
- 서로소 판별
