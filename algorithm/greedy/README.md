# Greedy

## 요약 설명
- 매 순간 최선의 선택을 고르면서, 최종의 해를 찾아가는 알고리즘
- DP(동적계획법)보다 빠르지만, 최적의 해를 항상 보장하지 않음

<br/>

## Greedy가 최적해를 보장하는 조건 

> **1. 탐욕 선택 속성**
>  - 이전의 선택이 이후의 선택에 영향을 주지 않아야 함

> **2. 최적 부분 구조**
>  - 부분 문제의 최적해가 전체 문제의 최적해를 만들어야 함

<br/>
